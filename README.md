# 奇门遁甲排盘引擎

> 基于 PyScript 的浏览器端排盘系统，无需服务器，打开网页即可使用。

## 在线使用

部署到 GitHub Pages 后，访问 `https://你的用户名.github.io/qimen-web/` 即可使用。

## 功能

- ✅ 完整的奇门遁甲排盘（地盘/天盘/人盘/神盘/旬空）
- ✅ 十二神将自动排布
- ✅ 三传推算
- ✅ 吉凶规则引擎（五级评分：大吉/吉/平/凶/大凶）
- ✅ 宫位总评与颜色标识
- ✅ 多占事类型支持（考试、走失、疾病、投资、工作、求财、婚姻、出行、竞争）

## 技术

- 前端：HTML + CSS + PyScript（浏览器端 Python）
- 排盘引擎：Python3 标准库（无外部依赖）
- 部署：GitHub Pages（免费静态托管）

## 本地使用

直接在浏览器打开 `index.html` 即可，无需安装任何环境。
